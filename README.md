# appliedSpatSync_concept
Data and code supporting "Toward an Applied Ecology of Spatial Synchrony" concept paper.
The repository contains:
1) conceptualFigure.r: code used to generate a conceptual illustration (Figure 1)
2) WOS_results_spatial_synchrony_20241125.csv: spreadsheet of results from a Web of Science search for "spatial synchrony" (all fields) conducted on 25 November 2024.
3) analysis.r: code used to analyze biblometric data. Totals the number of spatial synchrony journal articles published from 2010-2024 by journal title.
